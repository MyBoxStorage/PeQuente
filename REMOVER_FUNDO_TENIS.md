# Como Remover o Fundo Branco do Tênis

## Opções para Remover o Fundo Branco

### Opção 1: Remove.bg (Mais Fácil - Online)
1. Acesse: https://www.remove.bg/
2. Faça upload da imagem: `public/images/produtos/tenis-adidas-adizero-drive-rc-masculino-1.jpg`
3. O site remove automaticamente o fundo
4. Baixe a imagem PNG com fundo transparente
5. Renomeie para: `tenis-adidas-adizero-drive-rc-masculino-transparente.png`
6. Salve em: `public/images/produtos/`

### Opção 2: Photoshop / GIMP
1. Abra a imagem no Photoshop/GIMP
2. Use a ferramenta "Magic Wand" ou "Select Subject"
3. Selecione o fundo branco
4. Delete ou torne transparente
5. Salve como PNG (mantendo transparência)
6. Nome: `tenis-adidas-adizero-drive-rc-masculino-transparente.png`

### Opção 3: Canva (Gratuito)
1. Acesse: https://www.canva.com/
2. Crie novo design
3. Faça upload da imagem
4. Use a ferramenta "Remover fundo"
5. Baixe como PNG
6. Salve em: `public/images/produtos/tenis-adidas-adizero-drive-rc-masculino-transparente.png`

### Opção 4: Photopea (Editor Online Gratuito - Similar ao Photoshop)
1. Acesse: https://www.photopea.com/
2. Abra a imagem
3. Use "Magic Wand" para selecionar fundo branco
4. Delete o fundo
5. Exporte como PNG
6. Salve em: `public/images/produtos/tenis-adidas-adizero-drive-rc-masculino-transparente.png`

## Após Processar a Imagem

1. Salve a imagem PNG com fundo transparente em:
   ```
   public/images/produtos/tenis-adidas-adizero-drive-rc-masculino-transparente.png
   ```

2. O componente já está configurado para usar essa imagem
3. Se a imagem PNG não existir, ele automaticamente usa a JPG original como fallback

## Enquadramento

O componente já está configurado para:
- Enquadrar o tênis de forma harmoniosa
- Centralizar verticalmente e horizontalmente
- Ajustar tamanho automaticamente (máximo 600px de largura)
- Manter proporção original
- Adicionar sombra para destaque

## Dicas

- Mantenha a resolução alta (pelo menos 600x600px)
- Garanta que todo o fundo branco foi removido
- O tênis deve estar bem enquadrado na imagem
- Teste visualmente no site após salvar a imagem
